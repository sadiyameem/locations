# Areas of My Country

# Description
- This is a web-based project of different areas of my country (Bangladesh). It uses an expanding layout where each image response when clicked.

# Features
- Interactive panels that expands when clicked on
- Displays the titles and description of each image

# Tools
- HTML
- CSS
- JavaScript

# Link
- https://www.pexels.com/search/bangladesh%20areas/
- https://unsplash.com/s/photos/areas-in-bangladesh

Why I made this
- I made this project to help others learn more about my country through images and also improving my own understanding.
